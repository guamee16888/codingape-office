Codingape Office is a local AI coding worker console. It works only inside the project root you choose, generates evidence and diffs first, and writes only after approval.

3-minute Demo Send Feedback

Install Steps

  1. Download and open Codingape Office.app.
  2. Choose a local code project folder on first launch.
  3. Run the First Run safety check.
  4. Click "Run first task", review the diff, then decide whether to apply.

Safety Promise

No full-disk scan by default. Reads and writes are bound to the authorized project root. Project Root Guard blocks path traversal. Apply requires diff, verification, rollback, and human approval.

Beta Limits

This is a local Mac beta. The write runner is human-gated by default. Missing AIWC centralized logging is a warning only. Complex cross-repo tasks should be split into smaller tasks first.

Support And Feedback

If launch, port, Node/npm/Git, API key, apply, or rollback fails, generate a support bundle in Office and copy the diagnostic summary.
