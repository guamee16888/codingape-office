Coding猿 Office 是本地运行的 AI 程序员控制台。它只在你选择的 project root 内工作，先生成证据和 diff，确认后才允许写入。

安装说明

  1. 下载并打开 Coding猿 Office.app。
  2. 首次启动选择一个本地代码项目目录。
  3. 运行 First Run 安全自检。
  4. 点击“跑第一单”，审查 diff 后再决定是否 Apply。

安全承诺

不默认扫描全盘；所有读写绑定授权 project root；Project Root Guard 会阻断路径穿越；Apply 前必须有 diff、verification、rollback 和人工确认。

Beta 限制

当前是本地 Mac Beta，写入执行器默认 human-gated；AIWC 集中日志缺失只作为 warning；复杂跨仓库任务建议先拆小。

支持包与反馈

遇到启动、端口、Node/npm/Git、API Key、apply 或 rollback 问题，请在 Office 内生成支持包并复制诊断摘要。
