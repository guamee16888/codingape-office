Body:

```text
Hi HN,

I am building Codingape Office, an open-source local-first AI coding worker for macOS.

The goal is not to make a fully autonomous developer. The goal is to make AI-assisted code changes safer and more inspectable.

The workflow is:

choose project
-> build evidence
-> generate AI plan
-> generate patch
-> preview diff
-> run verification
-> Human Gate
-> Apply Gate
-> rollback snapshot
-> report

A few design principles:

- no full-disk scan by default
- user must choose the project folder
- no automatic code writes
- diff before write
- verification before apply
- explicit human approval
- rollback available
- API keys stay redacted from logs and support bundles
- supports Demo Only, BYO API keys, and local model endpoints

It is early, not a polished App Store release yet. I am looking for feedback from developers who care about local-first AI coding workflows and safety-gated patch application.

GitHub:
https://github.com/guamee16888/codingape-office
```
