# v0.1.0 - Safe Local AI Coding Worker For Mac

This is the first public open-source release of Codingape Office.

Codingape Office is a local-first AI coding worker for macOS. It helps developers safely test AI-assisted code changes with evidence, plans, diffs, verification, Human Gate, Apply Gate, rollback, and support bundles before project writes.

## Highlights

- Local project selection
- Project Root Guard
- Evidence Pack
- Model Provider Settings
  - Demo Only
  - BYO API Key
  - Local Model
- AI Plan Generator
- AI Patch Worker
- Unified diff validation
- Sandbox patch apply
- Verification runner
- Human Gate
- Apply Gate
- Rollback Manager
- Secret-redacted Support Bundle
- Mac App Shell and Mac App Store candidate pipeline
- AI patch evaluation fixtures and runner

## Safety Principles

- No full-disk scan by default
- No automatic code writes
- Diff before write
- Verification before apply
- Human approval before apply
- Sensitive files blocked
- Rollback available
- API keys redacted from logs and support bundles

## Status

This is an early open-source release.

It is useful for:

- local demos
- controlled AI patch experiments
- external pilot testing
- evaluating safety-gated AI coding workflows

It is not yet a polished App Store release.
