# Codingape Office v0.1.0

Initial open-source release candidate.

## Highlights

- Local office dashboard for macOS AI coding workflows.
- Project selection bound to a user-authorized project root.
- Evidence Pack, AI Plan, Patch Proposal, Diff Preview, Verification, Human Gate, Apply Gate, Rollback, and Company Report flow.
- Model Provider Settings for Demo Only, BYO API key, and local OpenAI-compatible endpoints.
- AI Patch Worker benchmark with small fixture projects and structured evaluation reports.
- Diff Contract Hardening for safer unified diff extraction, normalization, and formatting retry.
- Mac app build scripts for local beta, Developer ID distribution, and Mac App Store candidate preflight.

## Safety Defaults

- No automatic apply.
- No full-disk scan.
- Sensitive files are skipped.
- Project Root Guard blocks traversal and outside-root writes.
- Support bundles and evaluation reports must not include API keys or raw secrets.

## Known Limits

- App Store signing and upload still require the user's Apple Developer account materials.
- Real AI patch quality depends on the selected model provider and project context.
- External tester results should be collected separately; do not treat local benchmark success as production readiness.
